
# 🛒BuySphere - Your Ultimate Shopping Companion
A simple and responsive product listing web app built using HTML, CSS, and JavaScript. This project fetches real product data from an API and allows users to explore, search, and manage products easily.

---

## About
BuySphere is a front-end project created to practice JavaScript concepts like API integration, asynchronous programming, and array methods. It shows products in a clean UI and allows users to interact with them through different features.

---

## Features
- Search products by title
- Filter products by category
- Sort products by price or rating
- Add products to cart and manage quantity

---

## API Used
Fake Store API
https://fakestoreapi.com/

---

## Concepts Covered
- API Integration using fetch
- Asynchronous JavaScript
- Array methods (filter, map, sort)
- DOM Manipulation
- Event handling

---

## Technologies Used
- HTML
- CSS
- JavaScript (Fetch API)

---

## Project Goals
- Learn API integration using fetch
- Use array methods like filter, map, sort
- Build responsive UI
- Manage user interactions using JavaScript
